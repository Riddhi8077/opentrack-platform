
# OpenTrack

OpenTrack is a full-stack engineering dashboard designed for student developers preparing for global programs such as Google Summer of Code (GSoC), internships, and open-source contributions.

The platform helps users track applications, monitor contributions, and manage their technical growth in one centralized system.

---

## Key Features

- Program tracking for internships and student programs  
- Open-source contribution tracking  
- Personal learning roadmap  
- Notes and interview preparation  
- Secure authentication system  

---

## Tech Stack

**Frontend:** React, TypeScript  
**Backend:** Node.js, Express  
**Database:** MongoDB  
**Auth:** JWT
